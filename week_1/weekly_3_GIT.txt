Create a Word document for GIT commands used 


1. git config
command: git config –global user.name “[name]”  

command: git config –global user.email “[email address]”  

This command sets the author name and email address respectively to be used with your commits.

2.git init 
command: git init[repository name]

This command is used to start a new repository.

3.git clone
command: git clone [url]

This command is used to obtain a repository from an existing URL.

4.git add
command:git add [file]
This command adds a file to the staging area.

git add . :- this command is used to add all file at a time

5.git commit
command : git commit -m "[type the message]"
This command records or snapshots the file permanently in the version history.

6. git diff
command : git diff

This command shows the file differences which are not yet staged.

7.git status
command : git status

This command lists all the files that have to be committed.

8.git branch 
command : git branch

This command lists all the local branches in the current repository.

9.git push

command : git push [variable name] master

This command sends the committed changes of master branch to your remote repository.

10.git pull

command : git pull [repository link]

This command fetches and merges changes on the remote server to your working directory.

11. git branch

command : git branch 

this command give list of all branches.

12.git branch <branch name>

this command create a new branch with given name.

13. git checkout<branch name>

this command switch to specified branch name.

14.git log

Displays a log of commits, showing commit messages, authors, dates, and unique identifiers.

15. git merge<branch_name> 

Merges changes from the specified branch into the current branch.
