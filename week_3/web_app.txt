Create an Azure WebApp service, deploy a sample .Net Application on it


1.Search for "Web App" and select "Web App" from the search results.
2.Click on the "Create" button to start creating a new Web App.
3.Provide a name for your Web App.i.e ittweb
4.Create a new resource group i.e test-app
5.Select your preferred OS (Windows) and runtime stack (.NET 7).
6.Choose the region where you want to host your Web App.ie east us
7.Click on the "Review + create" button to review your settings. Review and create the Web App service
8.Click on the "Create" button to create the Web App service.
9.click deployment center 
10.select external git give url of your repository branch master 
11.click save
