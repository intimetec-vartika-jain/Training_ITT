Create a container registry in your Azure subscription using the Azure portal. 


1. Click on the "Create a resource" button (+) in the Azure portal.
2. In the search box, type "Container Registry" and select "Container Registry" from the suggested list.
3. Click on the "Create" button on the Container Registry blade.
4. provide the name for your container registry.i.e ittcontainer
5. Select the resource group where you want to create the registry or create a new one.i.e resource group
6. Choose the region where you want the registry to be located.
7. Select the SKU (pricing tier) for your registry. i.e basic
8. click review+create 
9. click create
