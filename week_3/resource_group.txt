Create a resource group and lock that resource group from azure cloud.



1.In azure portal search for resource group
2.Resource group window will appear click on create 
3.Provide name to resource group name i.e MyRg
4.Choose location i.e central india
5.click create and review
6.click create 
  
After that resource group will created after that add lock

1.on the left hand menu under setting click lock
2.on the lock screen click add
3.provide lock name i.e no delete
4.provide lock type i.e delete
5.click ok
